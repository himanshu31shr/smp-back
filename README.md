## 🎯 About

The backend API for the **Stock Market Portfolio (SMP)** platform — handles authentication, portfolio management, and market data aggregation.

Part of a **3-service architecture**:
- 📱 **[smp-front](https://github.com/himanshu31shr/smp-front)** — Angular frontend
- 🖥️ **[smp-back](https://github.com/himanshu31shr/smp-back)** — Node.js REST API (this repo)
- ⚡ **[smp-queue](https://github.com/himanshu31shr/smp-queue)** — Python queue worker

## 🚀 Getting Started

```bash
git clone https://github.com/himanshu31shr/smp-back.git
cd smp-back
npm install
npm start
```
